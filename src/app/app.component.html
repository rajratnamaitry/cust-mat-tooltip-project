Tooltip Tooltips display informative text when users hover over, focus on, or tap an element. Basic Tooltip Code Example 
        
<button mat-raised-button
       matCustomTooltip [toolTipMessage]="tooltipContent"
        aria-label="Button with a tooltip">
  Action
</button>
      

get tooltipContent(): string 

  return `<div class='cool-tooltip-content'>
    <h3>✨ Welcome!</h3>
    <p>This is a <b>stylish</b> tooltip with 
      <span class='highlight'>custom CSS</span>.
    </p>
  </div>`;
Tooltip Position Code Example 

<button mat-raised-button matCustomTooltip [toolTipMessage]="tooltipPostContent('Above')" matTooltipPosition="above">Above</button>
<button mat-raised-button matCustomTooltip [toolTipMessage]="tooltipPostContent('Below')" matTooltipPosition="below">Below</button>
<button mat-raised-button matCustomTooltip [toolTipMessage]="tooltipPostContent('Left')" matTooltipPosition="left">Left</button>
<button mat-raised-button matCustomTooltip [toolTipMessage]="tooltipPostContent('Right')" matTooltipPosition="right">Right</button>
Tooltip Delay Code Example 
            
              <button mat-raised-button
          matTooltip="Delayed tooltip"
          [matTooltipShowDelay]="1000"
          [matTooltipHideDelay]="2000">
          Show/Hide Delay
          </button>
Tooltip show/hide Code Example 

<button #coolTooltip="matCustomTooltip"
        mat-raised-button
        matCustomTooltip
        [toolTipMessage]="tooltipContent">
  Hover for Custom Tooltip
</button>

<button matButton
        mat-raised-button
        (click)="coolTooltip.show()"
        aria-label="Show tooltip on the button at the end of this section"
        class="example-action-button">
  Show Mat Custom Tooltip
</button>

<button matButton
        mat-raised-button
        (click)="coolTooltip.hide()"
        aria-label="Hide tooltip on the button at the end of this section"
        class="example-action-button">
  Hide Mat Custom Tooltip
</button>
